   ```bash

   ```

2. **Instal Dependensi:**

   ```bash
   composer install
   ```

3. **Pengaturan Lingkungan:**

   - Salin `.env.example` ke `.env`:

     ```bash
     cp .env.example .env
     ```

   - Sesuaikan file `.env` dengan kredensial database Anda.

   ```bash
   php artisan key:generate
   ```

   ```bash
   php artisan migrate
   ```

   ```bash
   php artisan serve
   ```
